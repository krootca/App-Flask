Flask - App

Página web en Python:

Esta página está totalmente construida en Python3, usando un modulo llamado Flask, gracias a ello, puedes ver esta página totalmente funcional donde el motor de plantillas es ninja2, y tambien puedes usar socket.io, por ejemplo, en las aplicaciones web para chats.

Nota: Jackeline si quieres ver la documentación de Flask en Python3 te dejo una documentación aquí

Modulos:

Todo esto usa ciertos modulos en python y la web en sí, trabaja con un motor de plantillas llamado ninja2, que básicamente ayuda a la conexión entre las variables de python y las etiquetas de html, pero los modulos principales son:
  • Flask
  • flask_socketio
  • random
  • time
  • os

Para la base de datos se utilizó:
  • sqlite3
  • re
  • random
  • time
